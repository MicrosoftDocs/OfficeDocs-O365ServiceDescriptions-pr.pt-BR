# <a name="insights"></a>Informações

O SharePoint Online é uma coleção de ferramentas e tecnologias baseadas na Web que ajudam sua organização a armazenar, compartilhar e gerenciar informações digitais. Criado com base no Microsoft SharePoint Server 2013, este serviço hospedado é ideal para trabalhar com projetos, armazenar dados e documentos em um local central e compartilhar informações. Os Recursos de Ideias de pesquisas a seguir permitem que as organizações usem as informações presentes em bancos de dados, relatórios e aplicativos empresariais para atender a necessidades específicas de seus negócios.
  
## <a name="business-intelligence-center"></a>Central de Business Intelligence

O site da central de Business Intelligence permite armazenar conexões de dados, pastas de trabalho, relatórios, scorecards e painéis em um local central seguro. Saiba mais sobre a [configuração de uma Central de Business Intelligence](https://go.microsoft.com/fwlink/p/?LinkId=270883).
  
## <a name="calculated-measures-and-members"></a>Medidas e membros calculados

Quando você cria relatórios de tabela e gráfico dinâmicos que usam dados armazenados nos Serviços de Análise do SQL Server, é possível criar medidas e membros calculados para esses relatórios. Saiba mais sobre [recursos de business intelligence nos Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271025).
  
## <a name="data-connection-library"></a>Biblioteca de conexões de dados

Uma biblioteca de conexões de dados é uma biblioteca especial do SharePoint que foi projetada para armazenar e gerenciar conexões de dados compartilhados. Elas podem incluir arquivos ODC que você cria por meio do Excel. Saiba mais sobre [bibliotecas de conexões de dados confiáveis no SharePoint Server 2013](https://go.microsoft.com/fwlink/p/?LinkId=271026).
  
## <a name="decoupled-pivottables-and-pivotcharts"></a>Tabelas e gráficos dinâmicos separados

Quando você criar um relatório de Gráfico Dinâmico, será criado automaticamente um relatório de Tabela Dinâmica correspondente. No Excel 2013, não é necessário exibir um relatório de tabela dinâmica na mesma planilha que o relatório de gráfico dinâmico correspondente. Você pode exibir apenas o relatório de gráfico dinâmico. Esse recurso é chamado *de tabelas dinâmicas e gráficos dinâmicos separados*. Saiba mais sobre [tabelas e gráficos dinâmicos](https://go.microsoft.com/fwlink/?LinkId=271027).
  
## <a name="excel-services"></a>Serviços do Excel

Os serviços do Excel permitem compartilhar pastas de trabalho com outras pessoas, interagir com relatórios, exibir e explorar informações em uma janela do navegador e atualizar dados. Os serviços do Excel também permitem compartilhar todos ou partes de pastas de trabalho com outras pessoas e controlar quais informações são exibidas nessas pastas de trabalho. No SharePoint 2016, os recursos dos serviços do Excel estão disponíveis por meio do Excel para a Web no servidor do Office Online. Os Serviços do Excel requerem o Servidor do Office Online no ambiente local. Saiba mais sobre os [Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271028).
  
## <a name="excel-web-access-web-part-for-sharepoint"></a>Web Part do Excel Web Access para SharePoint

Você pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint.
  
## <a name="field-list-and-field-support"></a>Lista de campos e suporte a campos

O Excel para a Web permite que você abra a lista de campos e o campo bem para relatórios de gráfico dinâmico e relatórios de tabela dinâmica que são exibidos nas janelas do navegador. Esse recurso facilita a alteração temporária das informações exibidas em um relatório de tabela dinâmica ou de gráfico dinâmico sem precisar abrir o Excel. Saiba mais sobre [Business Intelligence nos serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271029).
  
## <a name="filter-enhancements"></a>Aprimoramentos de filtro

É mais fácil criar segmentações de dados e vinculá-las a relatórios. Você pode ajustar o tamanho e a orientação de uma segmentação de dados. Você também pode formatar segmentações de dados aplicando estilos internos ou definindo seu próprio estilo.
  
## <a name="filter-search"></a>Pesquisa de filtro

Você pode pesquisar os resultados de filtro, o que facilita a navegação para um item específico de uma lista extensa. Saiba mais sobre o [uso de filtros e segmentações de dados](https://go.microsoft.com/fwlink/p/?LinkId=271030).
  
## <a name="odc-file-support"></a>Suporte a arquivos ODC

Os arquivos ODC permitem gerenciar, compartilhar e reutilizar conexões de dados com fontes de dados corporativos. As conexões de dados a determinadas fontes de dados podem ser configuradas e gerenciadas centralmente pelo pessoal de TI e ficar disponível para qualquer pasta de trabalho que consome arquivos ODC. Você pode manter conjuntos de arquivos ODC em locais centrais no farm do SharePoint e pode editar pastas de trabalho do Excel para usar esses arquivos.
  
## <a name="performancepoint-services"></a>Serviços do PerformancePoint

Não disponível para clientes do SharePoint Online. Os serviços do PerformancePoint permitem que os clientes do SharePoint Server 2013 criem painéis interativos que exibem indicadores de desempenho principais (KPIs) e virtualização de dados na forma de scorecards, relatórios e filtros. Saiba mais sobre os [Serviços do PerformancePoint](https://go.microsoft.com/fwlink/p/?LinkId=271032).
  
## <a name="power-bi-web-part"></a>Web Part do Power BI

O Power BI é um pacote de ferramentas de análise de negócios baseado em nuvem que permite que qualquer pessoa se conecte, visualize e analise dados com maior velocidade, eficiência e compreensão. Ele conecta os usuários a uma ampla variedade de dados ao vivo por meio de painéis simples, fornece relatórios interativos e visualizações atraentes para dar vida aos dados. Saiba mais sobre o Power BI em [Introdução ao serviço Power BI](https://go.microsoft.com/fwlink/?linkid=844671).
  
## <a name="power-pivot-for-excel-in-sharepoint"></a>Power Pivot para Excel no SharePoint

Com o Power Pivot para Excel, você pode realizar análises de dados avançadas e criar modelos de dados sofisticados. Você também pode combinar grandes volumes de dados a partir de várias fontes, analisar dados rapidamente e compartilhar as ideias de pesquisa. Com o SharePoint Online, você pode usar o Excel para a Web para exibir e interagir com gráficos, tabelas, relatórios de gráfico dinâmico e relatórios de tabela dinâmica em uma janela do navegador. Você também pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint. As folhas do Power Pivot devem ser criadas em Excel antes de serem visualizadas no SharePoint. Saiba mais sobre o uso do [Power Pivot em Excel](https://go.microsoft.com/fwlink/p/?LinkId=271036).
  
## <a name="power-view-for-excel-in-sharepoint"></a>Power View para Excel no SharePoint

 Com o Power View para Excel, você pode visualizar e interagir com dados modelados usando visualizações, animações e consultas inteligentes altamente interativas. Com o SharePoint Online, você usa o Excel para a Web para exibir e interagir com pastas de trabalho que contêm folhas do Power View em uma janela do navegador. Você também pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint. As folhas do Power View devem ser criadas em Excel antes de serem visualizadas no SharePoint. Saiba mais sobre o uso do [Power View em Excel](https://go.microsoft.com/fwlink/p/?LinkId=271035).
  
## <a name="quick-explore"></a>Exploração Rápida

A exploração rápida permite que as organizações selecionem um valor em um relatório de gráfico dinâmico ou de tabela dinâmica e vejam informações adicionais sobre esse valor. Por exemplo, se você está visualizando um relatório de vendas e seleciona um valor que representa os valores totais de venda, é possível usar a Exploração Rápida para exibir informações adicionais como valores de venda em diferentes categorias de produto e áreas geográficas. Saiba mais sobre a [Exploração rápida](https://go.microsoft.com/fwlink/p/?LinkId=271037).
  
## <a name="refresh-excel-workbooks-interactively"></a>Atualizar pastas de trabalho do Excel interativamente

Permite que você veja dados atualizados no Excel para pastas de trabalho da Web que estão conectadas a fontes de dados externas (por exemplo, bancos de dados MS SQL). Observe que você deve conectar suas pastas de trabalho a fontes de dados externas no Excel antes de exibi-las no Excel para a Web.
  
## <a name="sql-server-reporting-services-ssrs-integrated-mode"></a>Modo integrado do SQL Server Reporting Services (SSRS)

Quando os Serviços de Relatório do SQL Server 2012 são integrados ao SharePoint, você pode usar colaboração avançada do SharePoint e Recursos centralizados de gerenciamento de documentos para o conteúdo dos Serviços de Relatório do SQL Server 2012. Os relatórios são armazenados em bibliotecas de documentos do SharePoint, juntamente com outros relatórios, como o Excel para os arquivos da Web. As permissões do SharePoint podem ser usadas para controlar o acesso ao conteúdo dos Serviços de Relatório, e as pessoas podem iniciar o Designer de Relatórios, Designer de Modelos e Construtor de Relatórios em uma biblioteca de documentos. Saiba mais sobre o [servidor de relatório do Reporting Services](https://go.microsoft.com/fwlink/p/?LinkId=271039).
  
## <a name="sql-server-reporting-services-ssrs-web-part"></a>Web Part do SQL Reporting Services

A Web Part do SQL Server 2012 Reporting Services permite que as pessoas exibam um relatório do Reporting Services existente em sua própria Web Part em um site do SharePoint. Se o relatório do Reporting Services contiver parâmetros, as pessoas poderão conectar filtros a esse relatório usando as configurações da Web Part. Saiba mais sobre a [Web Part Visualizador de relatórios](https://go.microsoft.com/fwlink/p/?LinkId=271041).
  
## <a name="timeline-slicer"></a>Segmentação de linha do tempo

Uma segmentação de dados da linho do tempo é um controle que pode ser adicionado a um painel criado por meio do Excel. Um controle de linha do tempo permite que as pessoas exibam informações por um determinado período de tempo. Saiba mais sobre [Business intelligence nos Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271042).
  
## <a name="use-and-interact-with-excel-workbooks-containing-data-models"></a>Usar e interagir com pastas de trabalho do Excel que contêm modelos de dados

As Pastas de Trabalho do Excel às vezes contêm modelos de dados do PowerPivot que permitem executar poderosas análises de dados, lidar com volumes grandes de dados de várias origens, analisar dados rapidamente e compartilhar informações.
  
## <a name="user-defined-functions"></a>Funções definidas pelo usuário

As funções definidas pelo usuário (UDFs) dão a você a capacidade de usar fórmulas no Excel para as células da planilha Web chamar funções personalizadas escritas em código gerenciado. Os UDFs são um recurso de programação avançada que permite que sua empresa desenvolva e conecte o código personalizado no Excel para planilhas da Web. Você pode criar UDFs para chamar funções matemáticas personalizadas, obter dados de fontes de dados personalizadas em planilhas e chamar serviços Web dos UDFs.
  
## <a name="visio-services"></a>Serviços do Visio

O serviços do Visio é um aplicativo de serviço que permite que os usuários compartilhem\*e visualizem arquivos de desenho do Microsoft Visio\*(. vsdx) e desenho da Web (. vdw) do Visio 2010. O serviços do Visio não está mais disponível com o SharePoint Online. Os serviços do Visio continuarão disponíveis no SharePoint Server 2010, no SharePoint Server 2013 e no SharePoint Server 2016. Saiba mais sobre os [Serviços do Visio](https://go.microsoft.com/fwlink/p/?LinkID=271043).
  
## <a name="feature-availability"></a>Disponibilidade de recursos

Para exibir a disponibilidade de recursos nos planos do Office 365, nas opções autônomas e nas soluções locais, consulte [Descrição do serviço do SharePoint Online](sharepoint-online-service-description.md).
