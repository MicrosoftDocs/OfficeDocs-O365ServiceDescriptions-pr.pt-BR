# <a name="project-online-service-description"></a>Descrição de serviço do Project Online

Project Online é uma solução flexível de online para o gerenciamento de portfólio de projeto (PPM) e o trabalho diário. Fornecido por meio do Office 365, Project Online oferece recursos de gerenciamento de projeto poderosa para planejamento, priorização e gerenciamento de projetos e investimentos de portfólio de projeto — em praticamente qualquer lugar em praticamente qualquer dispositivo. Project Online pode ser usado por administradores, gerentes de portfólio e visualizadores, gerentes de projetos e recursos e os líderes de equipe e membros. 
  
O Project Online está disponível em três planos: Project Online Essentials, Project Online Professional e Project Online Premium.
  
Para comparar recursos entre planos, confira [Comparar planos do Project Online](https://products.office.com/en-us/project/compare-microsoft-project-management-software).
  
Você pode exportar, salvar e imprimir páginas nas Descrições de Serviço do Office 365. Saiba como [exportar várias páginas](https://go.microsoft.com/fwlink/?LinkId=403349).
  
## <a name="project-online-boundaries-and-limits"></a>Limites do Project Online

Há algumas limitações para o serviço do Project Online. Para mais detalhes, consulte [Limites do software Project Online](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489).
  
## <a name="data-backup"></a>Backup de Dados

 O Project Online se baseia no SharePoint Online e segue a mesma política de backup de dados do SharePoint. Para mais detalhes, consulte a seção Backup de Dados na [Descrição do Serviço do SharePoint Online](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx).
  
## <a name="use-of-sharepoint-online"></a>Usar o SharePoint Online

O serviço de projeto Online requer o uso do SharePoint Online, que será provisionada para você, como parte do serviço do Project Online. Direitos para a funcionalidade do SharePoint Online fornecidos com o Project Professional Online ou Premium estão limitados para armazenar e acessar os dados para suportar o Project Online.
  
## <a name="licensing-considerations"></a>Considerações sobre licenciamento

### <a name="internal-users"></a>Usuários internos

Nenhuma interação em um site do Project Online requer uma licença Project Online.
  
- O Project Online Essentials é o requisito mínimo.
    
- Para casos de uso mais avançados, os usuários precisam ter licenças apropriadas. Gerentes de projeto exigem, no mínimo, uma licença do Project Professional Online. Gerentes de portfólio exigem uma licença Premium on-line do projeto.
    
> [!IMPORTANT]
>  Project Online Essentials é uma licença do *complemento* do membro de equipe para clientes que tenham o Project Professional Online ou Premium on-line do projeto.<br/>Quando sua licença última do Project Professional Online ou Premium on-line do projeto expira, suas instâncias do Project Web App serão excluídas após 120 dias.<br/>Para o Project Online assinaturas de avaliação, suas instâncias do Project Web App serão excluídas 30 dias após o período de avaliação termina. 
  
### <a name="external-users"></a>Usuários externos

Licenciamento para usuários externos do Project Online segue a mesma política que para usuários internos. Nenhuma interação em um site do Project Online requer uma licença Project Online.
