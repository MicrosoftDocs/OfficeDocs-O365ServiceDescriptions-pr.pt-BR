# <a name="microsoft-365-government---how-to-buy"></a>Microsoft 365 governamental - como comprar

Em resposta às necessidades exclusivas e evolução do setor público dos Estados Unidos, a Microsoft criou ambientes específicos do Microsoft 365 governamental para nos clientes do setor público. Esta seção fornece uma visão geral de como os ambientes e obter mais informações sobre como comprar. É recomendável que você leia esta seção suplementar junto com as [Descrições do serviço do Office 365 ](../../office-365-service-descriptions-technet-library.md).
  
## <a name="whats-microsoft-365-government"></a>O que é o Microsoft 365 governamental?
<a name="TopOfPage"> </a>

Microsoft 365 Government é que um conjunto de recursos de software de produtividade, segurança e mobilidade nuvem adaptado especificamente para nós patrocinado para armazenar informações não classificadas controladas de prestadores de serviços e os órgãos governamentais. Entregue por meio de ambientes exclusivos que atendem aos requisitos de conformidade mais rigorosas, Microsoft 365 Government é uma oferta de nuvem para clientes do governo dos EUA que corresponda como bastante possível os recursos e capacidades do Microsoft cloud comercial ofertas de empresa.

## <a name="what-makes-microsoft-365-government-different-from-microsoft-365-enterprise-offerings"></a>O que torna Microsoft 365 governamental diferente do Microsoft 365 ofertas de empresa?
<a name="BM1_About"> </a>
  

Microsoft 365 governamental exclusivamente foi projetado para atender os requisitos de conformidade de nós os órgãos governamentais em quatro áreas principais:

**Comunidade exclusive.** Governo data centers estão disponíveis somente para agências governamentais ou departamentos — ou empresas comerciais que tenham sido autorizadas em espera e controlados pelo processo de informações em nome do governo dos EUA.

**Analisados pessoal.** Acesso ao conteúdo de cliente da sua organização é restrito aos filtrada equipe da Microsoft que é cidadãos dos Estados Unidos e passou nas verificações de plano de fundo necessários. Auditorias de terceiros. Toda a infra-estrutura é auditado por um auditor certificado de terceiros que fornece uma security assessment relatório ou certificação letra usada pelo agências federais para emitir uma autorização para operar (ATO). 

**Armazenamento de conteúdo.** Dados são armazenados dentro da área continental dos Estados Unidos com infra-estrutura compatível. A tabela a seguir resume 365 da Microsoft para ambientes de governamental norte-americana e comprometimentos.

## <a name="microsoft-365-for-us-government-eligibility-and-validation"></a>Microsoft 365 para validação de qualificação do governo dos EUA
<a name="BM2-customer"> </a>

Microsoft 365 governamental dos EUA está disponível para clientes do governo qualificados e organizações não-governamentais patrocinadas para manter ou processar informações controladas. Os critérios de qualificação para essas duas categorias de clientes é consistente em nuvem do governo da Microsoft.

Ofertas GCC e GCC alta estão disponíveis para qualquer cliente que é qualificado para a nuvem da Microsoft governamentais e o ambiente DoD destina-se a uso exclusivo do departamento de defesa dos EUA. Preço e disponibilidade do serviço são diferentes e GCC permanece o movimento herói para todos os clientes que não segurar FedRAMP alta ou DoD controlados informações não classificadas (CUI).

Um cliente do governo elegíveis definido por um o seguinte:

-   Uma agência federal, definida como um bureau, office, agência, departamento ou outra entidade governamental dos EUA
-   Uma entidade de estado/local, definida como um dos seguintes:
    1. Qualquer agência de um estado ou governamental local nos EUA
    
    2. Qualquer Condado dos Estados Unidos, borough, comunidade, cidade, município, cidade, township, Distrito propósito especial ou outro tipo semelhante de instrumentality governamental estabelecidos pelas leis do estado do cliente e localizados dentro jurisdição do estado do cliente limites geográficos e
    3. Do Distrito Federal, comunidade das Ilhas Mariana do Norte, Guam, Samoa Americana, as Ilhas Virgens dos Estados Unidos e Porto Rico
-   Uma entidade tribal, definida como uma entidade tribal nível federal reconhecida elegível para fundos e serviços do departamento do Interior dos EUA por seu status como uma tribo Indiana ou, em Alasca, um Vila nativa ou corporation Alasca regionais nativo

Organizações não-governamentais que mantenha os seguintes tipos de dados e podem fornecer as provas listadas qualificam para o processo de validação do governo de nuvem da Microsoft:

Uma entidade privada comercial com os dados que estão sujeito às normas. Tipos de dados do governo aceitos incluem: 
- Tráfego internacional no alerta (ITAR)
- Informações não classificadas controladas (CUI)
- Departamento de defesa não classificados controlados nucleares informações (UCNI)
- Departamento de energia (DoE) UCNI
- Informações de justiça criminal (CJI)
- Dados de nível de impacto do departamento de defesa
- Outros tipos de dados que exigem o Microsoft 365 governamental

Uma entidade comercial internacional pode qualificar, embora regulamentados dados possam ser exigidos para adquirir o serviço por meio de sua subsidiária dos EUA.

Prova de associação em um dos grupos listados acima serão necessária para acesso a ofertas governamentais do Microsoft 365.
